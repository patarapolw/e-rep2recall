# rep2recall-app

Repeat until you can recall, as a Desktop app.

Download: <https://github.com/patarapolw/rep2recall/releases>

![](http://g.recordit.co/6qMffliqFv.gif)

## Development mode

You can run in development mode with

```
cd app
yarn
yarn dev
```
